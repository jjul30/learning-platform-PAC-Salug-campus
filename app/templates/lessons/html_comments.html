HTML Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.

HTML Comment Tag

You can add comments to your HTML source by using the following syntax:

<!-- Write your comments here -->

Notice that there is an exclamation point (!) in the start tag, but not in the end tag.

Note: Comments are not displayed by the browser, but they can help document your HTML source code.

Example

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- This is another comment -->

Add Comments

With comments you can place notifications and reminders in your HTML code:

Example

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

Hide Content

Comments can be used to hide content.

This can be helpful if you hide content temporarily:

Example

<p>This is a paragraph.</p>

<!-- <p>This is another paragraph </p> -->

<p>This is a paragraph too.</p>

You can also hide more than one line. Everything between the <!-- and the --> will be hidden from the display.

Example

Hide a section of HTML code:

<p>This is a paragraph.</p>
<!--
<p>Look at this cool image:</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->
<p>This is a paragraph too.</p>

Hide Inline Content

Comments can be used to hide parts in the middle of the HTML code.

Example

Hide a part of a paragraph:

<p>This is a <!-- great text --> paragraph.</p>

Try It Yourself - Activity

Activity: Create an HTML document that includes:

  • A comment at the top explaining what the page is about
  • Comments before each section describing what it contains
  • A commented-out paragraph that is hidden from display
  • An inline comment within a paragraph

Meaning: HTML comments are essential for code documentation, making your HTML easier to understand and maintain. They help other developers (and your future self) understand your code structure and intentions.
