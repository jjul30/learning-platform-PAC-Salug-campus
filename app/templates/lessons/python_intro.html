Python Introduction

What is Python?

Python is a popular programming language known for its simplicity and readability.

Python Syntax

Python syntax is simple and easy to learn:

print("Hello, World!")

Variables

Python variables are created when you assign a value to them:

x = 5
y = "Hello"
z = 3.14

Data Types

  • int - Integer numbers
  • float - Decimal numbers
  • str - Text strings
  • bool - True/False values
  • list - Ordered collections

Functions

def greet(name):
    return f"Hello, {name}!"

message = greet("World")
print(message)

Loops

# For loop
for i in range(5):
    print(i)

# While loop
count = 0
while count < 5:
    print(count)
    count += 1
