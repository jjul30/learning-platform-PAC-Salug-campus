HTML Page Title

Every HTML document should have a page title to describe the meaning of the page.

The HTML <title> Element

The <title> element defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

The <title> element is required in HTML documents!

The content of a page title is very important for search engine optimization (SEO)! The page title is used by search engine algorithms to decide the order when listing pages in search results.

The <title> element:

  • defines a title in the browser toolbar
  • provides a title for the page when it is added to favorites
  • displays a title for the page in search engine-results

So, try to make the title as accurate and meaningful as possible!

Example

A simple HTML document:

<!DOCTYPE html>
<html>
<head>
  <title>A Meaningful Page Title</title>
</head>
<body>

The content of the document......

</body>
</html>

Tips for Good Titles

So, try to make the title as accurate and meaningful as possible. Here are some tips for creating good titles:

  • Never leave out the <title> element
  • Make the title describe the content and meaning of the page
  • Avoid one or two-word titles
  • Avoid extremely long title tags
  • Make it readable and human-friendly
  • Make every page title unique and different

Example

Good title examples:

<title>HTML Tutorial - W3Schools</title>
<title>Learn JavaScript - Complete Guide</title>
<title>Best Chocolate Cake Recipe - Easy and Delicious</title>

Example

Not so good title examples:

<title>HTML</title>
<title>Tutorial</title>
<title>This is a really really really really really really really really long title that should be shorter</title>

Dynamic Page Titles

You can change the page title dynamically using JavaScript:

Example

<script>
document.title = "New Page Title";
</script>

Practice Activity

Create meaningful page titles:

  1. Write a descriptive title for a personal portfolio page
  2. Create a title for a recipe website
  3. Write a title for a blog post about web development
  4. Consider SEO and user experience in your titles
